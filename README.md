## :earth_africa: Overview
This repository contains the **first binding global treaty proposal** to ensure AI and robotics **augment**—not replace—human labor, submitted to the **UN/ILO for 2026 adoption**. The framework mandates:

✅ **One-to-One Job Transition Guarantee** (ILO-aligned)
✅ **Automation Tax** funding universal reskilling
✅ **Transnational Oversight Board (TAEB)** with worker-majority representation
